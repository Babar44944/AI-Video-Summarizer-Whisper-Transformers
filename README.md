# 🎥 AI Video Summarizer

An AI-powered Video Summarization System that automatically converts videos into transcripts, summaries, topics, and timestamped reports using OpenAI Whisper and Transformer-based NLP models.

## 🚀 Features

* 🎤 Speech-to-Text with OpenAI Whisper
* 📝 Automatic Video Transcription
* 🤖 AI-Powered Summarization
* 📋 Short Summary Generation
* 📄 Detailed Summary Generation
* 📊 Executive Summary Reports
* 🎯 Topic Extraction
* ⏱️ Timestamp Generation
* 📁 TXT Export
* 📑 PDF Export
* 📦 ZIP Download Support
* 🌐 Interactive Streamlit Interface

## 🛠️ Technologies Used

* Python
* OpenAI Whisper
* Hugging Face Transformers
* DistilBART
* Streamlit
* PyTorch
* MoviePy
* FPDF2
* FFmpeg

## 📂 Workflow

